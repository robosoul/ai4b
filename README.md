# AI 4 B

A four-part foundation briefing on AI — made for Branka, open for anyone curious.

**Live:** https://robosoul.github.io/ai4b/

## Contents

- **Module 01 — Demystifying AI** — how we got here, what the words mean, how LLMs work
- **Module 02 — Responsible AI** — bias, hallucinations, ethics, privacy, case studies
- **Module 03 — AI Tools of the Trade** — prompting, the ecosystem, four tools up close
- **Module 04 — AI in Society** — agents, synthetic media, work, rights
- **Bonus — Bot. Copilot. Agent.** — a manager's briefing with real case studies and the reliability math

Each module is a self-contained reveal.js deck. Use arrow keys or swipe to navigate.
